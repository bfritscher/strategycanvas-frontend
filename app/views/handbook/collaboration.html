Collaboration

Sharing

Each canvas has two links: a view link and an edit link:

  • With the edit link the user has full access to edit the canvas.
  • With the view link a user can only see the canvas in a read-only state.

Realtime collaboration

Any edit done by an editor is immediatly shown to any connected user.

All users connected at the same time to the same canvas can chat together.

When more than one user is connected to the canvas a chat icon will appear on the top right. Clicking it will toggle the display of the chat window.

Users with the icon are viewers, the others are editors.
